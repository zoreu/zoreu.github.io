Gerador de Links direto do Google Drive para arquivos

Coloque a URL do Arquivo Compartilhada do Google Drive, por exemplo, https://drive.google.com/file/d/1kazyfB4JHoZSmczN-FBVXB4C8qN5b46G/view?usp=sharing:


Coloque sua Chave API (OPCIONAL SE ABAIXO DE 100MB):

 

Se o tamanho do seu arquivo for menor que 100 MB, a chave da API do Google Drive é opcional. Se o tamanho do seu arquivo for maior que 100 MB, em vez de fazer o download do arquivo, o Google exibirá um aviso "muito grande para o Google procurar vírus". Para ignorar o aviso, você deve aplicar a chave da API do Google Drive e inseri-la na caixa de entrada acima.
